# Netflix Data Cleaning & Visualization

## Overview
This project demonstrates data cleaning, preprocessing, and exploratory data analysis (EDA) on the Netflix Titles dataset.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Netflix Titles Dataset

## Data Cleaning Steps
- Handled missing values
- Removed invalid records
- Converted date columns to datetime format
- Created new features (year_added, month_added)

## Visualizations
- Movies vs TV Shows
- Top Countries
- Ratings Distribution
- Content Growth Over Years

## Key Insights
1. Movies dominate the Netflix catalog.
2. The United States contributes the highest number of titles.
3. TV-MA is the most common rating category.
4. Netflix content additions increased significantly after 2015.
